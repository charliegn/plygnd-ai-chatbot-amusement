Fun AI Chatbot for a fictious amusement park. Experimenting with Gerative AI on a Full Stack JavaScript stack

To setup the API:
Do to the fact using OpenAI can accrue finacial costs, you must bring your own OpenAI API key.

1. Create an OpenAI account and generate a API key
2. Duplicate and rename `.env.example` --to--> `.env`
3. Update the `OPENAI_API_KEY` env variable with your API Key

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
